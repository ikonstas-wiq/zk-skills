## Typography

**Font family: Roboto only.** No other fonts permitted.

| Element | Weight | Size | Notes |
|---------|--------|------|-------|
| Title slide heading | Roboto Medium | 34pt | Max 2 lines |
| Title slide subtitle | Roboto Light | 19pt | |
| Title slide date | Roboto Light | 12pt | |
| Headline | Roboto Medium | 22pt | Never change size/colour/position. Complete sentence, ideally 1 line, max 2. |
| Subtitle | Roboto Normal | 12pt | |
| Body content | Roboto Light | 10-14pt | Single font size per slide, minimum 8pt |
| Footnotes | Roboto Light | 8pt | Fixed size |

### Emphasis

- **Bold** key words or switch to Roboto Medium
- For stronger emphasis: bold + wiq Berry (`#913199`) colour

## Layout Rules

1. **Padding**: Text inside shapes must have 0.05in (0.13cm) padding on all sides
2. **Rounded rectangles**: All boxes must use rounded rectangle shapes — no sharp corners
3. **No-fly zone**: Keep content clear of the slide margins. Use `View > Guides > Show Guides` to see boundaries. Only footnotes may appear in the no-fly zone (sparingly).
4. **Headline stability**: If a headline is only 1 line, content below must NOT shift up. Anchor to slide guides.
5. **Whitespace**: Prefer visuals over dense text. Provide sufficient whitespace.
6. **Single font size per slide**: Don't mix body text sizes on the same slide (headlines/footnotes excluded).
